# Manuscript

This folder holds the methods/resource manuscript describing the Spatial
Foraging Platform: engineering, behavioral validation, electrophysiology /
sync validation, and analysis examples. Active drafting begins in
[Phase 8](../PROJECT.md#phase-8--manuscript-development).

## Source material

By the time Phase 8 starts, most of the manuscript content already exists
elsewhere in this repo:

- **Methods** — synthesized from [`../docs/`](../docs/) (architecture, sync, calibration, maintenance,
  user API).
- **Behavioral validation** — from the field test in Phase 5
  (see [`../docs/validation-plan.md`](../docs/validation-plan.md)).
- **Recording validation** — from HLAB's electrophysiology in Phases 3B and 5.
- **Resource statement** — from the [main README](../README.md) and the
  related-repository index.

## Planned contents (populated in Phase 8)

- `outline.md` — section assignments and target lengths.
- `figures/` — figure source files and exports.
- `sections/` — per-section drafts.
- `submission/` — final formatted manuscript and supplementary material.
