# Function Checks

> Status: stub. Drafted in Phase 0 (NTH).
>
> Per-module and system-level procedures to confirm the platform is healthy
> before an experiment.

## Module function check

_Step-by-step procedure to verify a single module: power-up indication,
communication, motor actuation, pellet detection, interaction sensing,
LED states._

## Base station function check

_Step-by-step procedure to verify the base station: discovery, time sync,
sync-output integrity._

## System function check

_End-to-end procedure exercising the full platform with all modules
attached._

## Pass/fail criteria

_What constitutes pass for each step. Where ambiguity is allowed and where
it is not._

## Tooling

_Required external tools (multimeter, scope, sync probe, etc.) and where
they are documented in the support-tool BOM._
